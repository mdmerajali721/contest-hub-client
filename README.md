## Key Features

1. Browse and explore admin‑approved contests without login
2. Secure contest registration with payment integration (Stripe)
3. Contest creators can add, edit, and manage contests
4. Declare winners and showcase achievements across the platform
5. Role‑based dashboards for Admin, Creator, and Normal Users
6. Leaderboard ranking users by number of contest wins
7. Live countdown timer with automatic contest end handling
8. Fully responsive design (mobile, tablet, desktop)
9. Dark/Light theme toggle (saved in localStorage)
10. Secure JWT‑based authentication with private routes
11. Submission system with modal‑based task uploads
12. User win‑percentage chart and profile customization
